# Tic Tac Toe Game

A simple Tic Tac Toe game built with Python's Tkinter library. This game allows two players to compete against each other on a colorful board. Players can enter their names at the start of the game.



## Features

- Colorful interface with a light blue background
- Players can enter their names
- Displays the winner or if the game is a draw
- Reset the game after a win or draw

## Requirements

- Python 3.x
- Tkinter (usually included with Python installations

## How to Play

- Upon starting the game, enter the names of Player 1 and Player 2.
- Players take turns clicking on the grid to place their symbols (X for Player 1 and O for Player 2).
- The game will announce the winner or indicate a draw when the game ends.
- The game will reset automatically, allowing for a new game to begin.
