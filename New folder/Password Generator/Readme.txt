## Password Generator


## Overview

This is a simple Python script that generates a random password of a specified length. The password includes a mix of lowercase letters, uppercase letters, digits, and special characters to enhance security.

## Features

- Generates a password with a customizable length.
- Ensures at least one character from each character set (lowercase, uppercase, digits, special characters) if the length allows.
- Randomly shuffles the characters in the password for added randomness.

## Requirements

- Python 3.x
- No external libraries required

## How to Use

- Open a terminal and navigate to the directory containing the script.
- Run the script :
- Enter the password length.
- The generated password will be displayed on the screen.

## Example
- Enter the password length: 12
- Generated password: aB3!dE$fGh1

## Notes
- The minimum length for a secure password is recommended to be at least 6-8 characters.
- The script ensures that the password contains at least one character from each category if the length is sufficient.
