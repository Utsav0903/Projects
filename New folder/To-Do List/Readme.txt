# To-Do List Application

## Overview

This To-Do List Application is a simple and user-friendly desktop application built using Python's Tkinter library. It allows users to manage their tasks efficiently by adding, removing, and clearing tasks from a list. The application features a clean and modern interface, making it easy to stay organized.

## Features

- Add Tasks : Users can input tasks into an entry field and add them to the list.
- Remove Tasks : Users can select a task from the list and remove it.
- Clear List : Users can clear all tasks from the list with a single click.
- User Notifications : The application provides warnings if the user tries to add an empty task or remove a task without selecting one.

## Requirements

- Python 3.x
- Tkinter (comes pre-installed with Python)

## Usage

- Launch the application, and you will see a title and an entry box.
- Type your task in the entry box and click the "Add Task" button to add it to your to-do list.
- To remove a task, select it from the list and click the "Remove Task" button.
- Click the "Clear List" button to remove all tasks from the list.


## How to Use the To-Do List Application

1. **Launch IDLE**:
   - Open IDLE on your computer. You can usually find it in your Python installation directory or by searching for "IDLE" in your applications.

2. **Open the Application Script**:
   - In IDLE, go to `File` > `Open...` and navigate to the directory where you saved the `todo_app.py` file.
   - Select the file and click `Open`.

3. **Run the Application**:
   - With the `todo.py` file open in IDLE, you can run the application by pressing `F5` or selecting `Run` > `Run Module` from the menu.
   - A new window will appear with the title "My To-Do List".

4. **Adding a Task**:
   - In the entry box at the bottom of the window, type the task you want to add (e.g., "Buy groceries").
   - Click the "Add Task" button.
   - If the task is added successfully, it will appear in the list above the entry box. If the entry box is empty, a warning message will prompt you to enter a task.

5. **Removing a Task**:
   - Click on a task in the list to select it.
   - Click the "Remove Task" button.
   - The selected task will be removed from the list. If no task is selected, a warning message will inform you to select a task.

6. **Clearing the List**:
   - To remove all tasks at once, click the "Clear List" button.
   - This action will delete all tasks from the list without any confirmation.

7. **Footer Message**:
   - At the bottom of the application, you will see a motivational message: "Stay organized!" to encourage you to keep track of your tasks.

8. **Closing the Application**:
   - You can close the application window by clicking the "X" button in the window's corner, just like any other application.

Now you can efficiently manage your tasks with this simple To-Do List Application!
